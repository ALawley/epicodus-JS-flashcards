JavaScript

The de facto language of the web, the only language understood by all modern web browsers.

Modulo

An operator that returns the remainder when one number is divided by another. (%).

Operators

Symbols used for arithmatic in the JavaScript language. (+, -, /, %, *, ^)

NaN

"Not a Number" Returned when dividing 0 by 0.

Variable Naming Conventions

First letter lowercase, first letter of additional words uppercase. (Lower camel case. All one word) Names should be contextually descriptive.

+=

A quick way to increment a number or variable. ("6 += 1" will return "7")

Function

The function takes an argument to perform an action.

Method

The method manipulates numbers and strings, and outputs a response.

Argument to a method

The section of a method inside parenthesis that typically specifies how a method is run.

String

A chunk of text held within quotation marks to differentiate it from variables.

Chaining methods

Allows user to run different methods sequentially separated by semi-colons in order to perform more complex operations.

Escape

By prefacing special characters that would otherwise get interpreted by JavaScript with a backslash you can use them safely.

Alert

An alert is a type of function that will print its argument, and doesn't take a response.

Commenting Code

Everything after a double-backslash until the end of line will not get read by the browser. Meant for note-making.

Booleans

A "true" or "false" response.

Undefined

Response when there is no output information, such as defining a new variable, or calling one that hasn't been set.

Parameters

A parameter is a variable that accepts an argument.

Return

Tells JavaScript to return the output of something.
